# repqr-onboarding-flow

Starter project for building the REPQR onboarding flow.

## Stack

- React
- TypeScript
- Vite

## Getting started

1. Install dependencies:
   - `npm install`
2. Start development server:
   - `npm run dev`
3. Build for production:
   - `npm run build`

## Project structure

- `src/App.tsx` main app shell
- `src/components/OnboardingFlow.tsx` onboarding flow component
- `src/styles.css` base styles
